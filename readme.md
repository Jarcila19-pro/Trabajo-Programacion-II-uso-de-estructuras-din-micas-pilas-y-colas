Es una aplicación desarrollada en Python como parte de las actividades académicas de la asignatura. El código implementa los temas y ejercicios propuestos por el profesor José Ubaldo.

## ¿Cómo se utiliza?

1. **Requisitos:**
   - Tener Python instalado en el computador.
   - (Opcional) Usar un entorno virtual para instalar dependencias.
2. **Ejecución:**
   - Abre una terminal en la carpeta del proyecto.
   - Ejecuta el archivo con el siguiente comando:
     ```
     streamlit run app.py
     ```
   - Sigue las instrucciones que aparecen en pantalla.
